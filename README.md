# Restful Sleep Tracker

## Setting project up for the first time
- Run `git clone git@github.com:EllisOllier/sleep-tracker.git`
- Open project with vscode
- Run `cd ./frontend`
- Run `npm install`
- New terminal or `cd ..`
- Run `cd ./backend`
- Run `npm install`
- You can now run `npm run dev` on the frontend and backend

## Running frontend

- `cd ./frontend`
- `npm run dev`

## Running backend
- `cd ./backend`
- `npm run dev`
